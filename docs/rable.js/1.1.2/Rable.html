    Rable

    Initial Library.

    Constructor

    new Rable(config)

    Function that is called upon initialization.

    Parameters:
    Name Type Description
    config object

    configuration applied by user.

    Properties
    Name Type Description
    port number

    The port that the application is running on.
    ssl object

    provide ssl certificate here if a secure webserver should be created (by documentation of nodejs.org). if no port was defined it will automatically be updated to 443.
    secondary boolean

    If set to true, no webserver will be launched. Usefull if you want to make use of the router only.
    root number

    amount of segments of the url to cut off. The original url will still be preserved req.initialUrl.
    matchByProxy boolean

    In case of rable running behind a proxy and if set to true, rable will match by url's given to the proxy, not the full url.
    template string | boolean

    The uuid of a template or set to false if no template should be loaded.
    theme string | boolean

    The uuid of a theme of set to false if no theme should be loaded.
    allowedSources string | boolean

    an array with allowed sources or set to false if no header should be passed. can also be updated with Rable.cors().
    skipThemeCompatibility boolean

    Skip the theme compatibility check if set to true.
    directoryIndexes array

    A list of directory indexes contained in an array. Default: index.rbl and index.html
    templatesDatabase string

    The location where you store your templates.
    themesDatabase string

    The location where you store your themes.
    Properties
    Name Type Description
    root string

    The root location of Rable.
    version string

    The current version of Rable.
    config object

    The configuration of Rable.
    server HTTP

    The http server. See NodeJS documentation.
    info object

    Information about the application.

    Properties
    Name Type Description
    get function

    Get the value of a property of the application info. (can be overwritten per route)
    set function

    Set the value of a property of the application info. (can be overwritten per route)
    smarttag SmartTag

    An extention module that handles global variables for SmartTag. See SmartTag for more.
    template Template

    The templating engine for rable.js. See Template for more.
    theme Theme

    The theme engine for rable.js. See Theme for more.

    use(path, middleware)

    Register a middleware.

    Parameters:
    Name Type Description
    path string

    NOT REQUIRED: In case it is defined, the middleware will only be triggered if the path matches the request.
    middleware function

    A function that will be called before calling the final action of the route. No paramenters need to be defined. req, res and next() will automatically be attached. It is required for next() to be called, instead to properly prevent the final action from being executed, updated res.finished to true.
